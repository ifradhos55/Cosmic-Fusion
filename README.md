## Technology Stack

This project is built on a **monolithic architecture** using standard ES Modules. It requires **zero build steps** (no Webpack/Vite needed).

| Component | Version | Role |
| :--- | :---: | :--- |
| **HTML5** | 5.2 | Structural foundation and DOM overlays. |
| **Three.js** | r160 | 3D scene graph, lighting, and camera management. |
| **WebGL** | 2.0 | Hardware-accelerated rendering context. |
| **Canvas API** | 2D | Runtime generation of planetary texture maps. |
